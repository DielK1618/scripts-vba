빠른 시작
corelib.xlam 로드부터 첫 번째 함수 호출까지

corelib.xlam은 Workbooks.Open 방식으로 로드합니다. AddIns 등록이 불필요하며, xlam 파일이 CWB(클라이언트 워크북)와 같은 폴더에 있어야 합니다.

📄 ThisWorkbook (cwb_01.xlsm) — Workbook_Open 표준 패턴
Private Sub Workbook_Open()
    On Error GoTo ErrHandler

    Dim strXlamPath As String
    Dim wbXlam      As Workbook

    strXlamPath = ThisWorkbook.Path & "\corelib.xlam"

    ' xlam 파일 존재 확인
    If Dir(strXlamPath) = "" Then
        MsgBox "corelib.xlam 을 찾을 수 없습니다.", vbCritical
        Exit Sub
    End If

    ' 이미 열려있는지 확인 (Workbooks 컬렉션에 xlam 미포함 → 오류 방식)
    On Error Resume Next
    Set wbXlam = Workbooks("corelib.xlam")
    On Error GoTo ErrHandler

    If wbXlam Is Nothing Then Workbooks.Open strXlamPath
    Exit Sub

ErrHandler:
    MsgBox "corelib.xlam 로드 오류: " & Err.Description, vbCritical
End Sub
💡 왜 Workbooks.Open? AddIns 등록 방식은 다른 파일에도 영향을 줍니다. Open 방식은 해당 프로그램 실행 중에만 xlam이 활성화되어 더 안전합니다.
📋
호출 규칙
Application.Run 으로 xlam 함수 호출하기
반환값 없는 Sub 호출
Application.Run "corelib.xlam!am_Core.DPUpdate_Off"
반환값 있는 Function 호출
Dim strPath As String
strPath = Application.Run( _
    "corelib.xlam!am_Core.XlamPath")
⚠️ Variant 배열 인수: Application.Run 으로 배열을 전달할 때, 수신 파라미터는 반드시 As Variant로 선언해야 합니다. As String()처럼 타입을 지정하면 Type Mismatch 오류가 발생합니다.
인수가 있는 호출 예시
' 인수 전달 — 콤마로 순서대로 나열
Dim strResult As String
strResult = Application.Run( _
    "corelib.xlam!am_Path.ReplacePath", _
    strPath, _               ' 인수 1: 변환할 경로
    ThisWorkbook.Path, _     ' 인수 2: CWB 폴더 경로
    ThisWorkbook.FullName)   ' 인수 3: CWB 전체 경로

Function ReplacePath(strPath, strCwbPath, strCwbFile, [blnChangeDrive], [blnAddDrive])

토큰이 포함된 경로를 실제 로컬 경로로 변환합니다. OneDrive, 네트워크 드라이브, 슬래시 혼용을 자동으로 처리합니다.

파라미터타입설명
strPathString변환할 경로 (토큰 포함 가능)
strCwbPathString호출한 CWB 폴더 경로
strCwbFileString호출한 CWB 전체 파일 경로
blnChangeDrive OptionalBooleanTrue: 드라이브를 CWB 드라이브로 강제 교체 (기본 False)
blnAddDrive OptionalBooleanTrue: 드라이브 없는 경로에 CWB 드라이브 추가 (기본 True)
반환: String — 변환된 경로 / 실패 시 ""
예시
Dim strResult As String

' 토큰 사용
strResult = Application.Run( _
    "corelib.xlam!am_Path.ReplacePath", _
    "{cPath}\DB\main.accdb", _
    ThisWorkbook.Path, ThisWorkbook.FullName)
' → "C:\MyApp\DB\main.accdb"

' 슬래시 혼용 경로 정규화
strResult = Application.Run( _
    "corelib.xlam!am_Path.ReplacePath", _
    "C:/Users/Name/Documents", _
    ThisWorkbook.Path, ThisWorkbook.FullName)
' → "C:\Users\Name\Documents"
